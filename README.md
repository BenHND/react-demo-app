# React test app

## Get Started

Upon install:

1. Run `yarn` or `npm install`

For development

1. Run `yarn start` in one terminal
2. Open http://localhost:3000 in a browser
3. Run `yarn start` in one terminal for running server (server port: 9876)

For production

1. Run `yarn build`
